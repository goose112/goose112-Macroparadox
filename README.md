# 🤖 Anime Paradox Automation Bot (Java Edition)

Проект по автоматизации игрового процесса в Tower Defense игре **Anime Paradox**. Бот использует компьютерное зрение для анализа экрана и управления игровыми механиками.

> ⚠️ **Статус проекта:** Разработка приостановлена (Archive). Проект содержит реализованную базу данных и ключевые алгоритмы автоматизации.

## 🌟 Ключевые возможности
*   **Определение локаций:** Идентификация текущего состояния (Лобби / Загрузка / Бой) на основе анализа изображений.
*   **Умная расстановка:** Автоматическая проверка персонажей на поле и их размещение.
*   **Взаимодействие с UI:** Поиск и клики по иконкам в лобби, автоматическое нажатие кнопки "Старт".
*   **Контроль камеры:** Программная корректировка угла обзора для точного позиционирования.

## ⚙️ Функционал в деталях
1.  **Распознавание образов:** Сравнение скриншотов игры с эталонными картинками кнопок и локаций.
2.  **Логика БД:** Сохранение параметров персонажей и логов действий бота в базе данных.
3.  **Автоматизация:** Полный цикл от входа в лобби до начала сражения без участия пользователя.
